# ZikaGame

ZikaGame é um jogo para conscientizar as pessoas dos perigos que o mosquito **Aedes Aegypti** com os vírus:

  - Dengue
  - Chikungunya
  - Zika

>Nosso trabalho é a conscientização das pessoas de maneira divertida e agradável.
